Every Event, Captured Automatically

The audit trail runs silently in the background, logging every aircraft movement, collision warning, operator action, and system event the moment it occurs — with no manual input required from your staff.

What Gets Logged

Complete Event Coverage

Every significant event in your hangar is automatically captured with a precise timestamp and full operator context.

Aircraft Movements

Every tug command, path taken, start position, and final placement is recorded — including which operator authorized the move.

Collision Warnings

All proximity alerts and collision warnings are timestamped with aircraft identities, hangar location, and how each event was resolved.

Operator Actions

Staff logins, command authorizations, manual overrides, and system configuration changes are all tied to individual user accounts.

System Health Events

Camera connectivity, tug battery levels, sensor calibration checks, and any system faults are continuously logged for maintenance records.

How It Works

Automatic, Secure & Always Available

No operator effort required — the system handles everything in the background.

Automatic & Secure Logging

The audit trail runs automatically in the background — no operator action required. Every event is written to a secure, append-only log the moment it occurs. Logs are stored locally and synced to the cloud, ensuring records are preserved even in the event of hardware failure.

Search, Filter & Export

Authorized personnel can search, filter, and export logs directly from the tablet interface, generating formatted reports for specific aircraft, date ranges, or event types in seconds.

Composite Safety Score

All collected event data — warnings issued, incidents avoided, operator response times, and system uptime — is processed into a single, standardized safety score. This gives insurance providers an objective, data-backed view of your FBO's operational risk profile, potentially supporting lower premiums.
